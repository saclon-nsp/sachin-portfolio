# Sachin Portfolio - Angular 20

## Run commands

Live Preview
https://saclon-nsp.github.io/sachin-portfolio/


```bash
cd ~/Downloads
unzip sachin-portfolio-ready.zip
cd sachin-portfolio-ready
npm install
npm start
```

Browser will open at:

```text
http://localhost:4200
```

## If Angular CLI issue comes

```bash
npx ng serve --open
```

## Build command

```bash
npm run build
```

Build output:

```text
dist/sachin-portfolio
```

## Important

Open `src/app/app.component.html` and replace this LinkedIn placeholder:

```html
https://www.linkedin.com/
```

with your real LinkedIn profile URL.
